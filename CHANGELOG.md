# Changelog

## [1.0.0] - 2025-01-01

### Added
- Interactive CLI with 7 categories and 50+ extensions
- `--list`, `--list --json`, `--category`, `--help` flags
- Direct install via `code` CLI
- `.vscode/extensions.json` generator
- Unit tests for loader, installer, args, prompt modules
- GitHub Actions CI on Node 16/18/20
- Issue templates: bug report, feature request, add extension, add category
- PR template
- Data-driven architecture: each category is its own JSON file in `data/categories/`
